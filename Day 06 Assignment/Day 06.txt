Q1 Why do you use position property in CSS?
__ Position property in CSS is used to you move the elements anywhere in the webpage. 
   The position property specifies the type of positioning method used for an element (static, relative, fixed,    absolute or sticky).

Q2 What is the difference between position property values : absolute / relative;?
__ Position: absolute places an element relative to its parent’s position and changing the layout around it whereas
   position: relative places an element relative to its current position without changing the layout around it.
